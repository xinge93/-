# 本地项目管理

项目管理页只在本机开放：`http://127.0.0.1:5173/#project-manager`。

## 启动

在项目根目录运行：

```bash
npm run dev
```

这个命令会同时启动网站和本地项目管理服务。上传的文件会保存到：

- `public/uploads/<项目编号>/`：项目封面与详情图片
- `public/data/managed-projects.json`：项目标题、说明、标签与图片地址

## 发布到线上

本地上传完成后，运行：

```bash
npm run build
```

随后将整个项目重新部署。线上页面会读取构建时包含的本地项目文件，但线上不会显示项目管理入口，也不能上传或修改文件。
